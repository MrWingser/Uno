# Ubuntu One for Mac

[Ubuntu One](http://one.ubuntu.com/) client for OS X.

### Requirements

Requires OS X Lion (10.7) and an [Ubuntu One][u1] account.

### Installation

[Download][download], unzip, and move to `/Applications` directory.
